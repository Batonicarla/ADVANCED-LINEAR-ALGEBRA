## Description
This project implements Principal Component Analysis (PCA) from scratch using eigenvalues and eigenvectors to reduce dimensionality of an African-based dataset.

## Requirements
- Python 3.x
- NumPy
- Pandas
- Matplotlib

## How to Run
1. Clone the repository
2. Open the notebook in Google Colab or Jupyter
3. Run all cells sequentially

## Dataset
The dataset contains missing values and non-numeric columns, which were preprocessed before PCA.
